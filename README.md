# 👥 AI Powered HR System

Upload multiple resumes (PDF) alongside a job description, and AI ranks and scores each candidate — matching skills, identifying gaps, and giving a shortlist recommendation.

## Tech Stack
Python, Streamlit, Groq (Llama 3.3), PyMuPDF, Plotly

## How to Run
1. `pip install streamlit groq python-dotenv pymupdf pandas plotly`
2. Add your Groq API key to `.env`: `GROQ_API_KEY=your_key`
3. `streamlit run app.py`
